# Client Detail Page

The client detail page is your complete view of a customer ‚Äî every job, invoice, payment, email, call, file, and activity in one place.

## Opening Client Details

Access the detail page by:
- **Clicking a client row** in the [client list table](/features/clients/client-list)
- **Clicking a client card** in [kanban](/features/clients/stages-pipeline) or mobile view
- **Searching** for a client and selecting from results
- **Following a link** from a job, invoice, or notification

## Page Layout

The client detail page has three main sections:

### 1. Summary Card (Top)

Quick overview with:
- **Client name** and company
- **Contact buttons** - call, email, SMS (one click)
- **Stage badge** - current pipeline status
- **Lifetime value** - total revenue from this client
- **Outstanding balance** - unpaid invoices
- **Edit button** - modify client info

### 2. Sidebar (Left)

Key information at a glance:
- **Contact details** - phone, email, website
- **Addresses** - property, billing, company
- **Preferred technicians**
- **Tags and custom properties**
- **Quick actions** - create job, send invoice, add note

### 3. Tabbed Content (Main)

Detailed sections for every aspect of the client relationship.

### Stage Management

Change the client's stage:
1. Click the stage badge
2. Select new stage from dropdown:
   - New Lead
   - Active Client
   - Inactive Client
   - Archived
3. Stage updates immediately
4. **Conversion tracking:** Moving to "Active Client" sets the `conversionDate` (first time only)

For visual pipeline management and drag-and-drop stage changes, use the [kanban board view](/features/clients/stages-pipeline).

## Payments Tab

Complete payment history with transaction details.

### Payment Records

**Columns:**
- Payment date
- Amount
- Payment method (Cash/Check/Credit Card/ACH/Stripe)
- Associated invoice(s)
- Reference number
- Recorded by (team member)
- Notes

**Filters:**
- Date range
- Payment method
- Invoice number

### Recording a Payment

1. Click **+ Record Payment**
2. Select invoice (or leave unassigned)
3. Enter amount
4. Choose payment method
5. Add reference number (check #, transaction ID)
6. Add notes (optional)
7. Click **Save**

Payment auto-applies to oldest unpaid invoice unless specified.

### Stripe Payment Methods

If Stripe is connected, see:
- Cards on file (last 4 digits)
- Default payment method
- Expiration dates
- **Add Card** button (client can save cards via portal)

## Files & Drive Tab

Central file storage for all client-related documents.

### File Organization

Files are organized by:
- **Type** (photos, PDFs, spreadsheets, documents)
- **Source** (uploaded, emailed, job-related)
- **Date uploaded**

### Uploading Files

1. Click **Upload** or drag files into the window
2. Select multiple files (shift-click or ctrl-click)
3. Files upload and appear in list

**Supported formats:**
- Images: JPG, PNG, GIF, WEBP
- Documents: PDF, DOCX, XLSX, TXT
- Other: ZIP, CSV

### File Actions

- **Preview** - view inline (images and PDFs)
- **Download** - save to your device
- **Share via Portal** - make visible to client
- **Delete** - remove file (with confirmation)
- **Add to Job** - attach to specific job

### AI File Chat (Beta)

Ask questions about uploaded documents:
1. Click **Chat with Files**
2. Ask questions (e.g., "What's the warranty period in this contract?")
3. AI searches all PDFs and documents for answers
4. Get instant summaries

## Tasks Tab

To-do list specific to this client.

### Task List

**Columns:**
- Task description
- Assigned to (team member)
- Due date
- Priority (High/Medium/Low)
- Status (Open/In Progress/Completed)
- Related job (optional)

### Creating a Task

1. Click **+ Add Task**
2. Enter description
3. Assign to team member
4. Set due date
5. Choose priority
6. (Optional) Link to a job
7. Click **Save**

Tasks appear in the assigned team member's task list and send notifications.

## Billing Tab

Manage Stripe payment methods for online payments.

### Payment Methods on File

**Displays:**
- Card brand (Visa, Mastercard, Amex)
- Last 4 digits
- Expiration date
- Default method (star icon)

### Adding a Payment Method

<Warning>
  **Client must add cards themselves** via the Client Portal. Admins cannot manually enter card numbers (PCI compliance).
</Warning>

To enable client card storage:
1. Enable **Client Portal** in Settings
2. Send portal invite to client
3. Client logs in ‚Üí Payment Methods ‚Üí Add Card
4. Card appears here after saved

### Charging a Saved Card

When creating an invoice:
1. Select client
2. Choose "Charge saved card" option
3. Select which card
4. Invoice auto-charges on save
