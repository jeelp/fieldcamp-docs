# API Reference

FieldCamp provides a REST API for integrating with your existing tools and workflows.

## Authentication

All API requests require a Bearer token. Generate an API key from **Settings â†’ API Keys**.

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.fieldcamp.ai/v1/jobs
```

## Base URL

```
https://api.fieldcamp.ai/v1
```

## Rate Limits

- 100 requests per minute per API key
- 10,000 requests per day per API key

## Endpoints

| Resource | Description |
|----------|-------------|
| `/v1/jobs` | Create, read, update jobs |
| `/v1/clients` | Manage client records |
| `/v1/invoices` | Invoice management |
| `/v1/team` | Team member operations |
| `/v1/schedule` | Scheduling operations |
